交互规则说明

入口与展示

  • 图片页操作区新增“从大货质检导入图片”。
  • 弹窗仅展示存在可导入图片的 SKU。
  • SKU Tab 展示 SKU 编码与变体值，数量较多时横向滚动。

选择与导入

  • 弹窗仅展示产品图片、测试图片。
  • 分类标题支持全选，图片支持多选、取消和预览。
  • 确认添加后关闭弹窗，可添加到 SPU 或当前页面已勾选 SKU。

去重与来源

  • 添加到 SPU 时按图片 ID 去重。
  • 添加到 SKU 时按图片 ID + SKU 维度去重。
  • 导入图片来源默认选中“大货质检”，可继续修改。
